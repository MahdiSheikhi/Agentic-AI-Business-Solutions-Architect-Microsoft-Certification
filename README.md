## Suggested study path

1. **Orient** — read the [Exam Overview & Study Plan](https://mahdisheikhi.github.io/Agentic-AI-Business-Solutions-Architect-Microsoft-Certification/AB-100-Study-Guide-Modules/00-overview.html) to understand the domains, weights, and question style.
2. **Learn** — work through the 25 modules in order, or study by tab in the interactive guide.
3. **Deepen** — for each domain, read the matching deep-dive notes and follow the Microsoft Learn source links for anything unfamiliar.
4. **Test** — take the in-guide quizzes (45 questions) and the 48-question practice exam; review every miss.
5. **Cram** — the day before, run the Quick-Review Cheat Sheet and the guide's Cram Sheet tab.
